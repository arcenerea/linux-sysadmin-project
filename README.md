 Proyecto de Administración de Sistemas Linux



Este es un proyecto sencillo para mostrar habilidades básicas como administradora de sistemas. Incluye un script para realizar una copia de seguridad automática.



\## Contenido



\- \*\*scripts/backup\_automatico.sh\*\*: Script que copia una carpeta (por defecto `Documentos`) a una carpeta `backup` con la fecha y hora.



\## Cómo usarlO



1\. Abre una terminal en Linux.

2\. Da permisos al script:

chmod +x scripts/backup\_automatico.sh

3\. Ejecútalo:

./scripts/backup\_automatico.sh



El backup se guardará en una carpeta llamada `backup` dentro de tu `/home`.
